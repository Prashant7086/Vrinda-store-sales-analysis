# 📊 Vrinda Store Sales Analysis (Excel Dashboard)

## 🚀 Project Overview
This project focuses on analyzing the sales data of **Vrinda Store** using Microsoft Excel.  
The goal is to extract meaningful insights, track performance, and support data-driven business decisions through an interactive dashboard.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-selling products  
- Understand customer purchasing behavior  
- Track revenue trends across time and locations  

---

## 🛠️ Tools & Technologies
- Microsoft Excel  
  - Pivot Tables  
  - Charts & Graphs  
  - Data Cleaning Techniques  
  - Dashboard Design  

---

## 📂 Dataset Description
The dataset includes:
- Order details  
- Product categories  
- Sales amount  
- Customer information  
- Order dates and locations  

---

## 📊 Key Features
- ✔ Cleaned and standardized raw data  
- ✔ Created multiple pivot tables for analysis  
- ✔ Built an interactive dashboard  
- ✔ Visualized sales trends and patterns  

---

## 🔍 Insights Generated
- 📈 Identified peak sales periods and high-performing months  
- 🛍️ Determined top-selling products and categories  
- 👥 Analyzed customer trends and purchasing behavior  
- 🏬 Compared performance across different store locations
